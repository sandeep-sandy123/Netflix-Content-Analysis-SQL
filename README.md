# Netflix Content Analysis — SQL

## Business Question
What does Netflix's content library reveal about
genre trends, regional production, and growth strategy?

## Tools Used
PostgreSQL | SQL

## Dataset
Netflix titles dataset — 8,800+ titles including
movies and TV shows across 190+ countries

## SQL Concepts Used
CTEs | Window Functions | Subqueries |
String Manipulation | CASE WHEN | Aggregation

## Key Findings
- Movies make up 69% of Netflix content vs 31% TV Shows
- USA, India and UK are top 3 content producing countries
- Drama and International Movies are the dominant genres
- Netflix content additions grew 3x between 2016 and 2020
- TV-MA is the most common rating across both content types

## Files
- netflix_queries.sql — all 13 SQL queries with comments

## How to Run
1. Import netflix dataset into PostgreSQL
2. Run netflix_queries.sql in sequence
3. Each query answers one business question
